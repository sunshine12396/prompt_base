> **Maestro Rule:** "If I can find this layout in a Tailwind UI template, I have failed."

## 1. Topological Betrayal (Phase 1)

Your primary goal is to betray the user's expectation of where elements "should" be.

### Radical Structures
- **Massive Typographic Hero**: Center the headline (300px+), build visual *behind* it.
- **Experimental Center-Staggered**: Every element (H1, P, CTA) has different horizontal alignment.
- **Layered Depth (Z-axis)**: Visuals that overlap text, challenging readability for artistic depth.
- **Vertical Narrative**: No "above the fold"; story starts immediately with vertical fragments.
- **Extreme Asymmetry (90/10)**: Compress everything to one extreme edge; 90% "dead space" for tension.

## 3. Visual Style Extremism

- **Geometry**: Stop using "soft lines" (rounded corners) by default.
  - **Sharp (0px-2px)**: Tech, Luxury, Brutalist.
  - **Soft (16px-32px)**: Social, Lifestyle.
- **Geometry Variety**: Every project should have different geometry.
- **Palette**: Deep Cyan/Fintech Blue is "Safe Harbor". Try Red, Black, or Neon Green.

---

## 4. Active Animation & Visual Depth

- **MANDATORY Reveal**: Staggered entrance for all main elements.
- **Micro-interactions**: Physical feedback (`scale`, `translate`, `glow-pulse`).
- **Spring Physics**: Organic movement, not linear.
- **Layering**: Overlapping elements, Parallax, Grain Textures.
